# What's this?

Get perfect shadows!

# Installation

`npm i sombritas --save`

Then...
```
import { sombritas } from "sombritas";

sombritas({
    shadow_type: "soft",
    padding: false
});
```
Sombritas supports 2 optios, both of which are optional:

* *shadow_type* - _hard | soft_ (Defaults to soft)
* *padding* - _boolean_ (Defaults to false)
